Want to make money for EHHOP?

If you ever shop from Amazon, it's easy to help EHHOP while shopping like you normally do! Just bookmark this page and use the link within the red box below before proceeding to shop at Amazon. All prices will appear the same to you on Amazon, but EHHOP will get a percentage of each purchase payment that you make.

↓ click within this box before you shop ↓

Some years we have been able to raise thousands of dollars thanks to this program. For a program like EHHOP, that runs mostly on volunteer labor and spends most of its funding on prescription medication for its patients, this is a lot of money.

If you see nothing in the red box above, you may have an ad blocker like AdBlock Plus installed. Try whitelisting this domain (ehhop0.appspot.com) and you will then be able to see the ad.
